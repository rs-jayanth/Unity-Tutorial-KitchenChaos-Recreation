This project is recreation of KitchenChaos Unity Game tutorial by code monkey for learning purpose.

https://unitycodemonkey.com/kitchenchaoscourse.php

Learned topics:

    * Character movements
    * Event Handling
    * Scriptable Objects
    * User Interface
    * Adding sounds
    * Binding Inputs
    * Particle effects
